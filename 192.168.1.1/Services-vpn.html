OpenVPN General:

Three auth methods are supported; pkcs12 (+dh on server), static and standard certs. Only enable MSS on one side of the link, fragment on both.

OpenVPN Server Additional Configuration:

To push routes to clients add 'push "route IP mask gateway"', to push DNS / WINS add 'push "dhcp-option DNS (or WINS) IP"' to the config.

Client connect directory:
When either USB or JFFS are mounted to /jffs, scripts will be called from /jffs/etc/openvpn/ccd/

OpenVPN Client Policy-based Routing:

Add IPs / NETs in the following format 0.0.0.0/0 to force clients to use the tunnel as the default gateway. Enter one IP / NET per line.

IP Address / Netmask:

Must be set when using DHCP-Proxy mode and local TAP is NOT bridged
